What's Running
About This Lab
GonzoTech is a fully self-hosted homelab running on TrueNAS (Dell Optiplex 3070 SFF). It powers a complete media, gaming, photo management, and development ecosystem — all accessible remotely via Cloudflare tunnels under gonzotech.us with zero port forwarding required.
🎬 Media Server
Jellyfin handles all media playback with Intel QuickSync hardware transcoding. Custom GonzoTech CSS branding applied.
watch.gonzotech.us
📡 ARR Stack
Automated media acquisition via Sonarr, Radarr, Prowlarr, and qBittorrent — all routed through NordVPN via Gluetun.
requests.gonzotech.us
📸 Photos
Immich handles self-hosted photo and video backup with an SMB external library for seamless NAS integration.
photos.gonzotech.us
🎮 Game Library
Romm manages a growing ROM collection — 2,300+ titles across 19 platforms, with IGDB metadata integration.
play.gonzotech.us
📊 Monitoring
Uptime Kuma keeps tabs on all services with real-time status checks and a public-facing status page.
status.gonzotech.us
💻 Dev Environment
Code-Server provides browser-accessible VS Code for remote configuration management and development work.
code.gonzotech.us

Network Architecture
All services are tunneled through Cloudflare Zero Trust — no ports exposed, no static IP required. Docker containers run in Dockge for easy stack management. DNS handled entirely through Cloudflare.
